# Rolls Car Rental - Premium Car Rental Website

A modern, responsive single-page car rental business landing page built with Bootstrap and jQuery.

## Features

- **Dynamic Hero Slider** - Owl Carousel with progress bar and smooth transitions
- **Services Section** - Modern card layout with hover animations
- **Stats Counter** - Animated number counters for business statistics
- **About Section** - Split layout with feature highlights
- **Fleet Gallery** - Isotope-powered filterable gallery with Magnific Popup
- **Team Section** - Team members with social media overlay on hover
- **Pricing Plans** - Four-tier pricing with featured plan badge
- **Testimonials** - Customer review carousel
- **Contact Form** - Validated contact form with Google Maps
- **Newsletter** - Footer newsletter subscription
- **Back to Top** - Smooth scroll back-to-top button
- **Preloader** - Animated page preloader

## Technologies

- HTML5 / CSS3
- Bootstrap 3
- jQuery
- Owl Carousel
- Isotope (Gallery filtering)
- Magnific Popup (Image lightbox)
- WOW.js + Animate.css (Scroll animations)
- Google Maps API
- Font Awesome

## Color Theme

- **Primary**: `#ffb74d` / `#ffc107` (Amber)
- **Accent**: `#45aed6` (Teal)
- **Dark**: `#031127` / `#121D29` (Navy)
